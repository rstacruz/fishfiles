Requires fish 2.6+.

```bash
brew install fish # macOS
sudo pacman -S fish # Arch Linux
```

Clone:

```bash
mkdir -p ~/.config
git clone https://github.com/rstacruz/fishfiles.git ~/.config/fish

```

In a new shell:

```bash
install-fisher
fisher update
update-settings

```
